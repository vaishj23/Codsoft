\# Task 1: Rule-Based Chatbot



\## Description

This project is a simple rule-based chatbot built using Python.

It responds to user inputs using predefined rules using if-else conditions.



\## Features

\- Greets the user

\- Answers basic questions

\- Handles unknown inputs

\- Exit command included



\## How to Run

1\. Open Command Prompt in this folder

2\. Run:

&nbsp;  python chatbot.py



\## Sample Output

User: hello

Chatbot: Hello! How can I help you?



User: bye

Chatbot: Goodbye! Have a nice day.
