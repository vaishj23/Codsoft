# Task 4 - Recommendation System

This project implements a simple movie recommendation system using content-based filtering.

## Description
The system recommends movies based on genre similarity using cosine similarity.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How to Run
1. Install dependencies:
   python -m pip install pandas scikit-learn

2. Run the program:
   python recommendation.py

3. Enter a movie name to get recommendations.

## Output
Displays a list of recommended movies based on similarity.
