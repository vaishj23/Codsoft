# Task 3 - Image Captioning

This project implements an Image Captioning system using a pre-trained deep learning model.

## Description
The program takes an input image and generates a natural language caption describing the image using a pre-trained BLIP image captioning model.

## Technologies Used
- Python
- PyTorch
- Transformers (Hugging Face)
- PIL

## How to Run
1. Install dependencies:
   pip install torch torchvision transformers pillow

2. Place an image named `sample.jpg` in the project folder.

3. Run:
   python image_captioning.py

## Output
The program prints a meaningful caption for the given image.
